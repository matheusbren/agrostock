# 🌱 AgroStock

Autor: Matheus de Queiroga Bren

Este projeto tem como objetivo implementar uma aplicação web responsiva para o controle de insumos agrícolas. O AgroStock permite ao usuário cadastrar, visualizar e gerenciar produtos como fertilizantes, sementes e defensivos, facilitando o controle de estoque e a organização das informações.
