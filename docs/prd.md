# 📄 PRD - AgroStock

## 👤 Identificação
- Autor: Matheus de Queiroga Bren
- Projeto: AgroStock – Controle de Insumos Agrícolas

---

## 📌 Descrição

O AgroStock é uma aplicação web que tem como objetivo auxiliar no controle de insumos agrícolas, permitindo o cadastro, visualização e gerenciamento de produtos como fertilizantes, sementes e defensivos.

O sistema resolve o problema da falta de organização no controle manual de insumos, facilitando o acesso às informações e melhorando a gestão agrícola.

---

## 👥 Atores do Sistema

- Visitante: pode visualizar os insumos cadastrados
- Usuário: pode cadastrar, editar e excluir insumos

---

## 📖 Histórias de Usuário

- Como usuário, quero cadastrar um novo insumo informando nome, categoria e quantidade, para manter o controle dos produtos.
- Como usuário, quero visualizar a lista de insumos cadastrados, para acompanhar o estoque.
- Como usuário, quero editar um insumo, para corrigir ou atualizar informações.
- Como usuário, quero excluir um insumo, para remover itens que não são mais utilizados.
- Como usuário, quero validar os dados do formulário, para evitar erros no cadastro.
- Como usuário, quero armazenar os dados no navegador, para não perder as informações ao recarregar a página.
- Como usuário, quero visualizar os insumos em formato de tabela ou cards, para facilitar a leitura.
- Como usuário, quero buscar endereço automaticamente pelo CEP, para facilitar o preenchimento do fornecedor.
- Como usuário, quero consumir dados de uma API, para tornar o sistema mais dinâmico.
